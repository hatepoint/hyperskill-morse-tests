Stage 2: Implementing Logic

Description

At this stage, the focus shifts to the core functionality of the app: the translation between text and Morse code. You will need to develop the code that maps characters to Morse code and vice versa. This involves coding the logic that responds to user inputs and translates them accurately.

Objectives

  1. Set up a map or equivalent data structure for mapping text characters to Morse code and Morse code to text.
  2. Implement the translation logic that uses the dictionary to convert input from the user into the desired output format.
  3. In morse, letters should be separated with space, and words with 3 spaces.
  4. If unknown symbol is entered, replace it with "?".
  5. Ensure the et_text changes triggers the conversion from text to Morse code and outputs the result to et_morse.
  6. Ensure the et_morse changes triggers the conversion from Morse code to text and outputs the result to et_text.
  7. Set action to clear both EditTexts after the bt_clear button is clicked.
